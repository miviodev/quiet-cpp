# Quiet но на C++
оригинал: https://github.com/miviodev/quiet

Эта программа позволит вам скрыть ненужные логи, но при ошибке выведет ее:
```bash
quiet gut clone # вместо git clone
```
вывод: sh: 1: gut: not found
## СБОРКА
### Под linux (unix):
просто ./compile.sh

после появится файл main.out это наш бинарник
### Под Windows:
для windows надо убрать коментарий на 16 строке и 15 строку закоментировать.

далее компилируете удобным для вас компилятором
